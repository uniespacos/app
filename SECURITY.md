# Security Policy

## Supported Versions

We actively support the following versions of UniEspaços with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | ✅ Yes             |

## Reporting a Vulnerability

We take security vulnerabilities seriously. If you discover a security vulnerability in UniEspaços, please help us by reporting it responsibly.

### 🔒 Private Disclosure Process

**Please DO NOT create a public GitHub issue for security vulnerabilities.**

Instead, please report security vulnerabilities via email or to the internal development team.

### 📋 What to Include

When reporting a vulnerability, please include:
- **Description**: A clear description of the vulnerability
- **Impact**: What an attacker could achieve by exploiting this vulnerability
- **Steps to Reproduce**: Detailed steps to reproduce the issue
- **Environment**: OS, Docker environment, etc.
- **Suggested Fix**: If you have ideas for how to fix the issue

### 🎯 Scope

This security policy applies to:
- UniEspaços core application (Laravel/React)
- Official Docker configuration
- Production environment configurations

*For detailed internal security reports, please check `docs/security-reports/`.*
